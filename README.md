# sqlpusher

	Csv reader --> pusher to SQL server:
	
	Usage of ./sqlpusher:

  -I CVS

      CVS file path/name

  -P password

      password

  -S server_name

      server_name[\instance_name]

  -U login

      login_id

  -d db_name

      db_name

  -m How many

      How many to insert at once (default 100)
